## Demo

## Run Locally

Clone the project

```bash
  git clone https://github.com/arisdanang/Submission-Belajar-Membuat-Aplikasi-Web-dengan-React
```

Go to the project directory

```bash
  cd Submission-Belajar-Membuat-Aplikasi-Web-dengan-React
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm start
```
